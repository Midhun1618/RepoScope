  <p><strong>Hybrid Repository Intelligence System</strong></p>

  <hr style="border: 1px solid #1e293b;" />

  <h2 style="color: #a5b4fc;">Overview</h2>
  <p>
    RepoScope is a full-stack web application that analyzes public GitHub repositories
    using a hybrid approach combining deterministic metrics and AI-powered interpretation.
  </p>
  <p>
    It evaluates repository health, risk, consistency, evolution trends,
    and generates strategic insights for long-term maintainability.
  </p>

  <hr style="border: 1px solid #1e293b;" />

  <h2 style="color: #a5b4fc;">Architecture</h2>
  <ul>
    <li><strong>Layer 1:</strong> Structural Parsing (GitHub REST API integration)</li>
    <li><strong>Layer 2:</strong> Rule-Based Heuristic Scoring Engine</li>
    <li><strong>Layer 3:</strong> LLM-based Strategic Interpretation</li>
    <li><strong>Layer 4:</strong> Interactive Visualization Dashboard</li>
  </ul>

  <hr style="border: 1px solid #1e293b;" />

  <h2 style="color: #a5b4fc;">Key Features</h2>
  <ul>
    <li>Repository metadata extraction and commit history analytics</li>
    <li>Health, Risk, Consistency, and Evolution scoring system</li>
    <li>Contributor and bus-factor risk analysis</li>
    <li>AI-generated development pattern & maintainability insights</li>
    <li>Interactive animated Bento-grid dashboard (React + Recharts + Framer Motion)</li>
    <li>CI/CD enabled deployment on Render</li>
  </ul>

  <hr style="border: 1px solid #1e293b;" />

  <h2 style="color: #a5b4fc;">Tech Stack</h2>
  <ul>
    <li><strong>Frontend:</strong> React, Recharts, Framer Motion</li>
    <li><strong>Backend:</strong> Node.js, Express</li>
    <li><strong>Database:</strong> Firebase Realtime Database (optional future extension)</li>
    <li><strong>APIs:</strong> GitHub REST API</li>
    <li><strong>AI:</strong> Large Language Model (LLM) Integration via OpenRouter</li>
    <li><strong>Deployment:</strong> Render (CI/CD enabled)</li>
  </ul>

  <hr style="border: 1px solid #1e293b;" />

  <h2 style="color: #a5b4fc;">Scoring Metrics</h2>
  <ul>
    <li><strong>Health Score</strong> – Structural and operational maturity</li>
    <li><strong>Risk Score</strong> – Maintainability and sustainability exposure</li>
    <li><strong>Consistency Score</strong> – Development regularity</li>
    <li><strong>Evolution Score</strong> – Growth and activity trajectory</li>
  </ul>

  <hr style="border: 1px solid #1e293b;" />
